# Konfiguracja MYSQL

Aby skonfigurować połączenie MYSQL należy utworzyć plik `.env` w głównym folderze.

Struktura pliku .env:

```
MYSQL_HOST=(TWÓJ HOST MYSQL)
MYSQL_DB=(NAZWA BAZY DANYCH MYSQL)
MYSQL_USER=(NAZWA UŻYTKOWNIKA MYSQL)
MYSQL_PASS=(HASŁO UŻYTKOWNIKA MYSQL)
```
